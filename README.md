<h2>Installing and configuring Oracle VM VirtualBox</h2>


<h2>Description</h2>

In this lab, I will be installing and configuring oracle VirtualBox which is a cross-platform virtualization software which will enable me to run Windows Server 2019 Datacenter edition as the operating system, and Microsoft Exchange as a mail and calendaring server.
<br />


<h4>The hash of any file I am downloading will be checked using Windows PowerShell before I install it, so I can guarantee it has not been compromised.</h4>
